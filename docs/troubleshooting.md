# Troubleshooting 🛠️

## Session and data issues
- If draft data disappears while creating a process:
  - Re-login and reopen `Home`
  - Reuse template if available
  - Save draft earlier in long forms

## Email not received
- Verify recipient email.
- Verify recipient is in current active group.
- Check spam/junk/quarantine.

## Login problems
- Confirm portal URL.
- Confirm account is enabled.
- Try clean browser session.

## Smart-ID / eID / Mobile-ID issues
- Verify recipient country/method combination.
- Verify personal data format (if non-anonymous).
- Verify mobile app/device readiness.

## Process actions disabled
- Completed/canceled/e-sealed processes are typically read-only.
- Some signer rows become locked by status progression.

> [!WARNING]
> If issue persists, provide process ID, timestamp, user role, and screenshot to support.
